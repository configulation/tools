## 1. 系统概述

### 1.1 系统用途

本系统用于管理生产线上的 **设备** 和 **工装** 的保养工作，包括：

- 登记和管理所有设备、工装的基本信息
- 制定保养计划，按周期自动提醒
- 执行保养操作并记录保养结果
- 查询和统计保养历史记录
- 支持 Excel 数据导入导出与备份

### 1.2 系统功能模块

| 序号 | 模块名称 | 功能说明 |
|:---:|---------|---------|
| 1 | 设备管理 | 添加、编辑、删除设备信息，查看保养状态 |
| 2 | 工装管理 | 添加、编辑、删除工装信息，跟踪使用次数 |
| 3 | 保养计划 | 按月查看保养日历，了解每天的保养任务 |
| 4 | 保养执行 | 执行具体的保养操作，支持扫码快捷操作 |
| 5 | 保养记录 | 查询历史保养记录，查看统计分析 |
| 6 | 基础数据 | 维护线别储位、类别、保养项目等基础信息 |
| 7 | 数据导入导出 | Excel导入导出、数据备份与恢复 |

## 2. 系统启动与版本说明

### 2.1 打开系统

在主程序的菜单中，点击 **"设备工装保养"** 即可打开本系统。

### 2.2 版本检查机制

系统启动时会 **自动检查版本**：
- 系统会连接数据库，对比当前程序版本和数据库要求的最低版本
- 如果版本一致或更高，正常进入系统
- 如果版本过旧，会弹出 **红色提示框**，告知当前版本和要求版本，然后 **自动关闭系统**

> **重要提示**：
> - 系统运行过程中，每 **1小时** 会自动检查一次版本
> - 如果管理员在后台更新了版本要求，即使不重启程序，系统也会在下一次检查时自动拦截
> - 遇到版本过期提示时，请 **联系管理员** 获取最新版本的程序

### 2.3 版本过期时的提示画面

当版本过期时，系统会弹出如下提示：

```
┌──────────────────────────────────────┐
│        版本过期 - 禁止访问            │
│                                      │
│  当前程序版本过旧，已被禁止使用！       │
│                                      │
│  当前版本：1.0.0                      │
│  要求版本：2.0.0                      │
│                                      │
│  请联系管理员获取最新版本后再使用。      │
│                                      │
│              [ 确定 ]                 │
└──────────────────────────────────────┘
```

点击「确定」后，系统会自动关闭。

## 4. 设备管理

### 4.1 功能说明

设备管理模块用于管理所有生产设备的基本信息和保养状态。

### 4.2 设备列表页面

进入设备管理后，可以看到所有设备的列表，表格包含以下列：

| 列名 | 说明 |
|------|------|
| 设备ID | 设备的唯一编号，如 "SMT-001" |
| 线别储位 | 设备所在的产线和位置 |
| 类别 | 设备的大类，如 "贴片机" |
| 子类别 | 设备的小类，如 "高速贴片机" |
| 周期(天) | 保养周期，每隔多少天保养一次 |
| 计划日期 | 下次保养的计划日期 |
| 状态 | 当前设备状态 |

**设备状态说明**：

| 状态 | 含义 |
|------|------|
| 新购 | 新购入的设备，尚未正式使用 |
| 正常使用 | 设备正在正常运行中 |
| 保养中 | 设备正在进行保养 |
| 维修中 | 设备出现故障正在维修 |

### 4.3 颜色标识

列表中的每一行会根据保养日期显示不同的底色：

| 颜色 | 含义 |
|------|------|
| 🔴 浅红色底 | **已过期** — 保养日期已过，需要立即安排保养！ |
| 🟡 浅黄色底 | **即将到期** — 保养日期在7天以内，请尽快安排 |
| 🟢 浅绿色底 | **正常** — 保养日期还很充裕，无需紧急处理 |

### 4.4 操作按钮

#### 4.4.1 新增设备

1. 点击 **「新增」** 按钮
2. 在弹出的编辑窗口中填写以下信息：
   - **设备ID**（必填）：输入设备的唯一编号，如 "SMT-001"
   - **线别储位**（必填）：从下拉列表选择设备所在位置
   - **类别**（必填）：从下拉列表选择设备大类
   - **子类别**：选择类别后，子类别下拉列表会自动更新，选择对应的子类别
   - **保养周期（天）**（必填）：输入数字，表示每隔多少天保养一次，如 "15"
   - **计划日期**：选择下次保养的日期
   - **状态**：选择设备当前状态（默认为"正常使用"）
   - **操作员ID**：输入负责该设备的操作员工号
   - **保养项目**：点击 **「选择保养项目」** 按钮，勾选该设备需要做的保养项目
   - **备注**：可选填备注信息
3. 填写完成后点击 **「保存」**
4. 系统会验证数据，如果有错误会弹出提示，请根据提示修改
5. 保存成功后会提示"添加成功"，列表自动刷新

#### 4.4.2 编辑设备

1. 在列表中 **选中一行设备**（点击该行即可选中）
2. 点击 **「编辑」** 按钮
3. 在弹出的编辑窗口中修改需要变更的信息
4. 注意：**设备ID不可修改**（灰色不可编辑状态）
5. 修改完成后点击 **「保存」**

#### 4.4.3 删除设备

1. 在列表中 **选中一行设备**
2. 点击 **「删除」** 按钮
3. 系统会弹出确认对话框："确定要删除选中的设备吗？"
4. 点击 **「确定」** 执行删除，点击 **「取消」** 放弃删除

> ⚠️ **注意**：删除操作不可恢复，请谨慎操作！

#### 4.4.4 执行保养

1. 在列表中 **选中一行设备**
2. 点击 **「保养」** 按钮
3. 系统会打开保养执行窗口（详见第7章）
4. 完成保养后，设备的计划日期会自动更新

#### 4.4.5 刷新

点击 **「刷新」** 按钮可以重新加载最新的设备数据。

### 4.5 搜索功能

在页面顶部的搜索框中输入关键字，系统会 **实时过滤** 设备列表，支持按以下字段搜索：
- 设备ID
- 线别储位
- 类别
- 子类别

清空搜索框后会自动显示所有设备。

### 4.6 选择保养项目

在新增/编辑设备时，可以点击 **「选择保养项目」** 按钮，弹出保养项目选择窗口：

1. 窗口中会列出所有可用的保养项目（来自基础数据中的"设备"类保养项目）
2. **勾选** 需要的保养项目（可多选）
3. 点击 **「全选」** 可以一次选中所有项目
4. 点击 **「清空」** 可以取消所有选中
5. 选好后点击 **「确定」**，保养项目会显示在文本框中

## 5. 工装管理

### 5.1 功能说明

工装管理模块用于管理所有工装的基本信息、使用次数和保养状态。

### 5.2 工装列表页面

表格包含以下列：

| 列名 | 说明 |
|------|------|
| 工装编码 | 工装的唯一编号 |
| 线别储位 | 工装所在的产线和位置 |
| 类别 | 工装的大类 |
| 子类别 | 工装的小类 |
| 使用次数 | 显示格式为 "当前次数/总限制次数"，如 "1500/200000" |
| 计划日期 | 下次保养的计划日期 |
| 状态 | 当前工装状态 |

### 5.3 颜色标识

| 颜色 | 含义 |
|------|------|
| 🔴 浅红色底 | **已过期或使用次数已满** — 保养日期已过，或使用次数已达到限制 |
| 🟡 浅黄色底 | **即将到期** — 保养日期在7天以内 |
| 🟢 浅绿色底 | **正常** — 保养时间和使用次数都在安全范围 |

### 5.4 操作按钮

#### 5.4.1 新增工装

1. 点击 **「新增」** 按钮
2. 在弹出的编辑窗口中填写以下信息：
   - **工装编码**（必填）：输入工装的唯一编号
   - **线别储位**（必填）：从下拉列表选择工装所在位置
   - **类别**（必填）：从下拉列表选择工装大类
   - **子类别**：选择类别后，自动加载对应的子类别列表
   - **工单号**：输入关联的工单号
   - **工单数量**（必填）：输入工单的数量（数字）
   - **拼料数量**（必填）：输入拼料的数量（数字）
   - **刮刀数量**（必填）：输入刮刀的数量（数字）
   - **使用次数**：当前已使用次数（可手动输入或自动计算）
   - **总使用次数**：使用次数上限
   - **保养间隔**：保养间隔说明，如 "200000次"（默认值）
   - **计划日期**：下次保养的日期（默认为30天后）
   - **状态**：选择工装当前状态
   - **保养项目**：选择需要做的保养项目
   - **备注**：可选填
3. 点击 **「保存」**

#### 5.4.2 自动计算使用次数

在新增/编辑工装时，有一个很实用的功能——**自动计算使用次数**：

1. 输入 **工单数量**、**拼料数量**、**刮刀数量**
2. 点击 **「计算使用次数」** 按钮
3. 系统会自动计算：**使用次数 = 工单数量 ÷ 拼料数量 ÷ 刮刀数量**
4. 计算结果会自动填入"使用次数"字段

> 例如：工单数量 = 10000，拼料数量 = 5，刮刀数量 = 2  
> 计算结果：10000 ÷ 5 ÷ 2 = **1000 次**

#### 5.4.3 编辑工装

1. 在列表中 **选中一行工装**
2. 点击 **「编辑」** 按钮
3. 修改需要变更的信息（工装编码不可修改）
4. 点击 **「保存」**

#### 5.4.4 删除工装

1. 选中一行工装 → 点击 **「删除」** → 确认删除

#### 5.4.5 执行保养

1. 选中一行工装 → 点击 **「保养」** 按钮
2. 打开保养执行窗口（详见第7章）

#### 5.4.6 搜索

搜索框支持按工装编码、线别储位、类别、子类别进行实时过滤。

## 7. 保养执行

### 7.1 功能说明

保养执行模块用于实际执行保养操作，有两种进入方式：

- **方式一**：从左侧导航菜单点击「保养执行」进入 → 显示 **待执行任务列表**
- **方式二**：从设备管理/工装管理中选中设备/工装 → 点击「保养」按钮 → 直接进入 **保养表单**

### 7.2 待执行任务列表（从菜单进入）

进入后会看到所有需要保养的任务列表：

| 列名 | 说明 |
|------|------|
| 类型 | "设备"或"工装" |
| 编号 | 设备ID或工装编码 |
| 线别储位 | 所在位置 |
| 类别 | 类别和子类别 |
| 计划日期 | 保养计划日期 |
| 状态 | "已过期 X天"/"今天"/"X天后" |
| 操作 | 点击「执行保养」按钮 |

**任务颜色说明**：

| 颜色 | 含义 |
|------|------|
| 🔴 浅红色底 | 已过期的保养任务（优先处理！） |
| 🟡 浅黄色底 | 3天内即将到期的保养任务 |

#### 操作步骤

1. 查看任务列表，找到需要保养的设备或工装
2. 点击该行右侧的 **「执行保养」** 按钮
3. 在弹出的保养表单中填写保养信息
4. 保存后任务会从列表中消失（已完成）

### 7.3 扫码快捷操作

在待执行任务列表页面，点击 **「扫描编码」** 按钮：

1. 弹出输入框
2. **输入或使用扫码枪扫描** 设备ID或工装编码
3. 按回车键或点击 **「确定」**
4. 系统自动识别是设备还是工装，直接打开保养表单
5. 如果编码无法识别，会提示"未找到该设备或工装"

> 💡 **小技巧**：使用扫码枪对准设备/工装上的条码标签，可以快速扫描编码，省去手动输入的时间。

### 7.4 保养表单（填写保养信息）

无论是从任务列表还是设备/工装管理进入，都会打开保养表单：

| 字段 | 说明 | 是否必填 |
|------|------|:-------:|
| 设备ID/工装编码 | 自动填入（灰色不可编辑） | — |
| 保养时间 | 默认为当前时间，可修改 | ✅ |
| 操作员 | 输入执行保养的操作员姓名 | ✅ |
| 保养项目 | 选择本次执行的保养项目 | ✅ |
| 保养结果 | 选择"正常"或"异常" | ✅ |
| 下次保养间隔天数 | 仅工装保养时显示，输入天数 | ✅（仅工装） |
| 备注 | 可选填备注，如异常情况说明 | ❌ |

#### 操作步骤

1. 确认设备ID/工装编码正确
2. 确认或修改保养时间
3. **填写操作员姓名**（必填！）
4. 点击 **「选择保养项目」**，勾选本次执行的保养项目，点击确定
5. 选择保养结果：**正常** 或 **异常**
6. （如果是工装保养）输入下次保养间隔天数
7. 如有需要，在备注中填写补充信息
8. 点击 **「保存」**
9. 系统提示"保养记录已保存"表示操作成功

> ⚠️ **注意**：保存后，该设备/工装的下次保养日期会自动根据保养周期重新计算。

## 8. 保养记录

### 8.1 功能说明

保养记录模块用于查询、查看和统计历史保养记录。

### 8.2 筛选条件

| **查看详情** | 查看选中记录的完整详情 |
| **统计分析** | 按当前筛选时间范围进行统计 |
| **导出** | 将查询结果导出（功能开发中） |

### 8.4 记录列表

| 列名 | 说明 |
|------|------|

| 操作员 | 执行保养的人员 |
| 结果 | "正常"或"异常" |
| 备注 | 备注信息 |

**行颜色说明**：

| 颜色 | 含义 |
|------|------|
| 🔴 浅红色底 + 深红文字 | 保养结果为 **异常** 的记录 |

### 8.5 查看详情

   - 类型（设备/工装）
   - 编号
   - 保养时间
   - 保养项目（完整列表）

   - 备注

统计内容包括：

### 9.5 复制功能

复制功能可以快速创建相似的数据：
1. 选中一条数据
2. 点击 **「复制」**
3. 系统自动在编码和名称后添加"_Copy"和"_副本"后缀
4. 修改需要变更的字段
5. 点击 **「保存」** 即可创建新数据

> 💡 **建议**：如果某个类别或保养项目暂时不需要使用，建议 **停用** 而不是删除，以保留历史记录。

## 10. 数据导入导出

### 10.1 功能说明

数据导入导出模块支持：
- 从 Excel 文件导入数据
- 将数据导出为 Excel 文件
- 全量数据备份与恢复
- 下载导入模板

### 10.2 导入数据

支持导入 Excel 文件（.xlsx 和 .xls 格式）的数据类型：

| 按钮 | 功能 |
|------|------|
| 导入设备数据 | 从 Excel 文件批量导入设备信息 |
| 导入工装数据 | 从 Excel 文件批量导入工装信息 |
| 导入保养计划 | 从 Excel 文件导入保养计划 |
| 导入保养记录 | 从 Excel 文件导入历史保养记录 |

**导入操作步骤**：

1. 点击对应的 **「导入」** 按钮
2. 在弹出的文件选择对话框中，选择 Excel 文件
3. 点击 **「打开」**
4. 系统自动读取并导入数据
5. 导入成功后会提示"成功导入 X 条数据"
6. 如果导入失败，会提示错误原因

> ⚠️ **重要提示**：
> - 导入前请确保 Excel 文件的格式与模板一致
> - 建议先下载模板，按照模板格式填写数据
> - 导入数据会新增到现有数据中，不会覆盖原有数据

### 10.3 导出数据

| 按钮 | 功能 |
|------|------|
| 导出设备数据 | 将所有设备信息导出为 Excel 文件 |
| 导出工装数据 | 将所有工装信息导出为 Excel 文件 |
| 导出保养计划 | 将保养计划导出为 Excel 文件 |
| 导出保养记录 | 将保养记录导出为 Excel 文件 |

**导出操作步骤**：

1. 点击对应的 **「导出」** 按钮
2. 在弹出的保存对话框中，选择保存位置
3. 文件名会自动生成（如"设备数据_20260206.xlsx"）
4. 点击 **「保存」**
5. 导出成功后会提示"导出成功"

### 10.4 下载导入模板

1. 点击 **「下载模板」** 按钮
2. 选择模板文件的保存位置（选择文件夹）
3. 点击 **「确定」**
4. 系统会在该文件夹下生成所有类型的导入模板文件
5. 按照模板格式填写数据后，即可使用导入功能

### 10.5 数据备份

数据备份功能可以将所有数据一次性导出到一个文件夹中：

1. 点击 **「数据备份」** 按钮
2. 选择备份保存的文件夹
3. 系统会自动创建一个以时间戳命名的子文件夹（如 `Backup_20260206143000`）
4. 在该文件夹中生成 4 个 Excel 文件：
   - 设备数据.xlsx
   - 工装数据.xlsx
   - 保养计划.xlsx
   - 保养记录.xlsx
5. 备份完成后提示保存位置

> 💡 **建议**：定期（如每周或每月）进行一次数据备份，防止数据丢失。

### 10.6 数据恢复

从备份文件夹中恢复数据：

1. 点击 **「数据恢复」** 按钮
2. 选择之前备份的文件夹（如 `Backup_20260206143000`）
3. 系统会自动读取文件夹中的 Excel 文件并导入
4. 恢复完成后提示恢复的数据条数

> ⚠️ **注意**：恢复操作会将备份数据导入系统，请确认选择的是正确的备份文件夹。

## 12. 常见问题与处理

### Q1: 打开系统后提示"版本过期"怎么办？

**答**：说明您的程序版本已过旧，请联系管理员获取最新版本的程序。安装新版本后重新打开即可。

### Q2: 打开系统后提示"版本校验失败，无法连接数据库"怎么办？

**答**：可能是网络连接问题或数据库服务器异常。请检查以下几项：
1. 电脑是否已连接公司网络
2. 联系IT管理员确认数据库服务器是否正常
3. 等待几分钟后重试

### Q3: 使用过程中系统突然关闭了？

**答**：系统每1小时会自动检查版本。如果管理员更新了版本要求，系统会弹出提示后自动关闭。请联系管理员获取最新版本。

### Q4: 新增设备/工装时提示"设备ID已存在"怎么办？

**答**：说明该编号已经被使用了。请检查编号是否输入有误，或使用一个新的唯一编号。

### Q5: 下拉列表中没有我需要的选项怎么办？

**答**：需要先在 **「基础数据」** 模块中添加。例如：
- 缺少线别储位 → 基础数据 → 线位储位 → 新增
- 缺少类别 → 基础数据 → 类别 → 新增
- 缺少保养项目 → 基础数据 → 保养项目 → 新增

### Q6: 如何批量导入设备数据？

**答**：
1. 进入「数据导入导出」模块
2. 先点击「下载模板」获取标准模板
3. 按模板格式在 Excel 中填写设备数据
4. 点击「导入设备数据」，选择填写好的 Excel 文件

### Q7: 保养完成后，计划日期没有更新？

**答**：保养操作必须通过系统中的「保养执行」功能完成，手动修改设备信息中的日期无法自动生成保养记录。请使用正确的保养流程。

### Q8: 如何查看某台设备的保养历史？

**答**：进入「保养记录」模块，在关键字搜索框中输入设备ID，即可查看该设备的所有保养历史记录。

### Q9: 工装的"使用次数"怎么计算？

**答**：在工装编辑界面：
1. 输入工单数量、拼料数量、刮刀数量
2. 点击「计算使用次数」按钮
3. 系统自动计算：使用次数 = 工单数量 ÷ 拼料数量 ÷ 刮刀数量

### Q10: 数据丢失了怎么办？

**答**：如果之前做过数据备份，可以通过「数据导入导出」→「数据恢复」功能恢复。如果没有备份，请联系IT管理员查看是否有其他方式恢复数据。
