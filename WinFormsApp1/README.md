# Excel数据导入系统

这是一个基于SunnyUI框架开发的高端Excel数据导入系统，支持多种数据库类型，提供直观的字段映射和数据导入功能。

## 功能特点

### 数据库配置模块

- 支持多种数据库类型选择（MySQL、PostgreSQL、SQL Server、Oracle）
- 数据库连接字符串配置（支持明文/密文显示）
- 连接测试功能
- 数据库表结构加载

### Excel字段映射模块

- Excel文件上传与预览
- 数据库表字段显示
- 直观的字段映射配置
- 数据类型转换设置
- 必填字段标记

### 数据导入模块

- 导入进度实时显示
- 导入结果统计与错误反馈
- 批量数据处理

### 附加功能

- 配置模板保存与加载
- 高端UI设计，支持多种主题

## 技术栈

- .NET Framework
- SunnyUI框架
- 支持多种数据库连接

## 使用说明

1. 在"数据库配置"标签页中设置数据库连接信息
2. 在"字段映射"标签页中上传Excel文件并配置字段映射
3. 在"导入数据"标签页中执行数据导入操作

## 注意事项

- 使用前请确保已安装相应数据库驱动
- 对于大文件导入，建议分批处理
